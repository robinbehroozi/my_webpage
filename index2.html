Image Gallery

This is where the image gallery will be in Wordpress. To create the gallery, we will have a loop here that will show blog posts that are in a certain category. That way, we will be able to differential between this blog post (which will be a gallery), and the lower blog posts, which will be actually blog posts. This loop will look for blog post that are in the 'gallery' category. The lower loop will look for posts in the 'blog' category. In Wordpress, I used to WP_Query object to do this. Here is what the code looks like. Please image it inside php blocks.

$args = array( 'category_name' => 'gallery' ); $the_query = new WP_Query( $args ); if ($the_query->have_posts()): while ($the_query->have_posts()): $the_query->the_post(); php the_content(); endwhile; endif;
